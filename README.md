# DecodeLabs Internship

## Week 1: Responsive Frontend Interface

This project was developed as part of the DecodeLabs Full Stack Development Internship Week 1 task. The website is built using HTML5, CSS3, and JavaScript, focusing on responsive design principles, semantic HTML structure, Flexbox, Grid layouts, and basic interactivity. Features include a responsive navigation menu, dark mode toggle, smooth scrolling, feature cards, contact form, and a modern user interface that adapts seamlessly across desktop, tablet, and mobile devices.

### Technologies Used
- HTML5
- CSS3
- JavaScript

### Features
- Responsive Design
- Mobile Navigation Menu
- Dark Mode Toggle
- Smooth Scrolling
- Interactive Contact Form
- Semantic HTML Structure
- Flexbox and CSS Grid Layouts

Developed for educational and portfolio-building purposes under the DecodeLabs Internship Program.
